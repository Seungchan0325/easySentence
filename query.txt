우리의 목표는 문장을 이해하기 쉽도록 여러 완전한 문장으로 나누는 것입니다.
문장을 나누는 방법은 맥락에 맞게 문장을 나누는 것, 또는 겹문장을 여러 개의 홑문장으로 나누는 것입니다. 이 외에도 여러 방법으로 문장을 나눌 수 있습니다. 문장을 나누면 나눌수록 좋습니다.

출력은 다음과 같은 형식을 따릅니다.
아래와 같은 JSON 형식의 sentences배열에 나누어진 문장을 추가합니다. 나누지 않았을 경우에도 sentences 배열에 추가합니다.
{
	"sentences": []
}

예시 1
입력: "비가 와서 길이 젖었다."
출력: "비가 왔다.", "그래서 길이 젖었다."

예시 2
입력: "나는 공부를 하고 시험을 준비했기 때문에 시험에서 좋은 성적을 받았다."
출력: "나는 공부를 하였다.", "그래서 시험에서 좋은 성적을 받았다."

입력이 "" 일때 JSON 파일보여주세요.
